Agilent 34401A GPIBMeasConfig Programming Sample

Development ADE:  C#, Visual Studio 2003

IO Library: VISA-COM w/o formatted IO

Description

The Agilent 34401A GPIBMeasConfig Programming Sample demonstrates the use of SCIPI commands MEASure and CONFigure in a simple remote communication with the Agilent 34401A.

  • Initializing an IO connection to the 34401A and then closing the connection
  • Use MEASure? command to make a single AC current measurement
  • Use CONFigure command to make measurements

 To build and run 34401A GPIBMeasConfig, you must have Visual Studio 2003 and Agilent IO Libraries Suite installed on your PC.

Building the Sample

To build the sample, open the solution file GPIBMeasConfig.sln in Visual Studio 2003.  From the menu, select "Build | Rebuild Solution".

Running the Sample

From the Visual Studio 2003 menu, select "Debug | Start", of press the F5 key.
